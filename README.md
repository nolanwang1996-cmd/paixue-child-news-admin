# 儿童新闻电台后台原型

这是一个可直接部署的静态后台原型，当前数据保存在浏览器 `localStorage` 中。

## 本地预览

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173
```

## 默认登录

```text
账号：editor
密码：demo123
```

## 部署方式

推荐任选一种：

- Vercel：适合快速生成公网预览地址。
- Netlify：适合静态站托管和后续表单/函数扩展。
- GitHub Pages：适合代码托管后自动发布。

当前项目已经包含 `vercel.json` 和 `netlify.toml`，可以直接作为静态站发布。
